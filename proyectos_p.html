Proyectos de Programación.

BlackJack

Se trata de un juego de cartas comocido como BlackJack, el cual consiste en sumar un valor total de 21 con dos cartas o más sin pasar dicho número.

El proyecto fue realizado con HTML, CSS y JavaScript en un entorno virtual desarrollado en FLASK.

El código fuente se encuentra disponible en mi repositorio de GitHub: BlackJack GitHub

To-Do

Éste es un proyecto que enlista tareas por hacer y las guarda en una base de datos relacionada al usuario. Cuenta con un login y un registro para la administración de las tareas.

El proyecto fue realizado con HTML, CSS y Python en un entorno virtual desarrollado en Django.

El código fuente se encuentra disponible en mi repositorio de GitHub: To-Do GitHub

Cyberstore

Este es un proyecto de una tienda virtual llamada Cyberstore, la cual contiene varios módulos para la gestion de productos, gestion de usuarios, panel administrador, carrito de compras productos en descuento y pasarela de pagos.

El proyecto fue realizado con HTML, CSS y JavaScript en un entorno virtual desarrollado en FLASK.

El código fuente se encuentra disponible en mi repositorio de GitHub: Cyberstore GitHub
